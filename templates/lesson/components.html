Alerts

To make an alert box, just use a block-level element (like a div) and add the .alert-box class. You can use the .secondary, .success, and .alert classes to style alerts (just like buttons).

To add a close button to your alert, add the data-alert attribute, and an anchor with the class .close.

Labels

Foundation also lets you easily create labels, useful inline styles that can serve as tags or call our certain sections.

All you need is a span tag with the class .label. Like alerts, labels can be styled with the .secondary, .success, and .alert classes, but they can also be styled with .radius and .round classes for different shapes.

Progress Bars

To create a progress bar, create a div.progress, and within that a span.meter. The meter is what shows the actual progress, and its width will reflect the amount of progress made. The easiest way to update it is to change the style="width:%" on the element itself.

You can change the color and shape of the progress bar with the familiar .secondary, .success, .alert, .radius and .round classes. If you need to control the width of the progress bar, the .small-# and .large-# classes are available.
