Dropdowns & Splits

Making dropdown buttons is super easy. Just taking an existing button, and add .dropdown to it. This styles the button properly, but you'll have to use the dropdown plugin for the dropdown content itself.

Similarly, if you want a split button (the left side is a button and the right side is a dropdown), just add .split to an existing button, and inside of the button, add an empty span tag.

Try making a dropdown or a split button in the editor.
