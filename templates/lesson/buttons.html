That's not the only kind of button you can make, though. Depending on your use case, you might need some different sizes, shapes, and colors.

All that changes is the class name. .tiny, .small, .medium (the default) and .large change size, while .secondary, .success, and .alert change color.

You can also check out .radius, .round, and .disabled for more variations. In general, you can attach the .button class (and its associated styles) to any a, button, or input tags.

Button Groups

Sometimes though, you'll want to use groups of buttons (to offer several actions as a bar). All of the same button classes still work, you just have to wrap everything in a ul.button-group tag. You can change the size and color of each individual button, but add .round or .radius to the ul.

By default, each button will be the standard button width. If you want to change the widths, you can add a .even-# class. .even-2 will make each button wide enough to have 2 buttons take up the entire width, .even-3 will do the same for 3 buttons, all the way up to .even-8.

If you want a longer bar made of several button groups, just wrap your button groups in a div.button-bar.

Dropdowns

Making dropdown buttons is super easy. Just taking an existing button, and add .dropdown to it. This styles the button properly, but you'll have to use the dropdown plugin for the dropdown content itself.

Similarly, if you want a split button (the left side is a button and the right side is a dropdown), just add .split to an existing button. Using split buttons requires the dropdown plugin as well.
